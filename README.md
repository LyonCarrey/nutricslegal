# frametrics-legal

Public legal pages for **身衡 · Frametrics** (App Store privacy policy).

This repository is intentionally separate from the private app source (`LyonCarrey/MyBody`).

## Privacy Policy URL (for App Store Connect)

Target URL after Pages is live:

**https://lyoncarrey.github.io/frametrics-legal/**

Content lives on branch `gh-pages` (`index.html`).

## Before App Store submit

1. Edit `index.html` on `main` **and** `gh-pages` (or edit on `main` then copy) — replace  
   `REPLACE_WITH_YOUR_SUPPORT_EMAIL@example.com` with your real support email.
2. In GitHub → this repo → **Settings → Pages**:
   - Source: **Deploy from a branch**
   - Branch: **`gh-pages`** / **`/ (root)`**
   - Save, wait until the green check appears
3. Open the URL above and confirm the page loads (not “Site not found”).

If Pages stays on “errored” / “Site not found”, open **Actions** and check whether workflows are stuck queued (new accounts sometimes need a browser visit to enable Actions / Pages).

## Scope

Privacy policy HTML only. No app source.
