# frametrics-legal

Public legal pages for **身衡 · Frametrics** (App Store privacy policy URL).

This repository is intentionally separate from the private app source repo.

## Live URL (after GitHub Pages is enabled)

`https://lyoncarrey.github.io/frametrics-legal/`

Use that URL as the **Privacy Policy URL** in App Store Connect.

## Before submit

1. Edit `index.html` and replace `REPLACE_WITH_YOUR_SUPPORT_EMAIL@example.com` with your real support email (same as ASC support contact).
2. Confirm GitHub Pages is on: **Settings → Pages → Deploy from a branch → `main` / `/ (root)`**.

## Scope

- Privacy policy HTML only.
- No app source, no food catalog, no internal research docs.
